## üåê&nbsp; The Ubiquity of JavaScript

Neob code is transpiled to **JavaScript**, the most ubiquitous programming language of the modern computing era. Present in browsers, servers, mobile devices, desktop applications, and even embedded systems, JavaScript offers unparalleled reach: **write once, run anywhere**. Its massive ecosystem ‚Äî with millions of libraries, tools, and mature runtimes ‚Äî enables complete applications to be built, distributed, and executed with remarkable ease, from frontend to backend.

JavaScript combines a high-level execution model, automatic garbage collection, dynamic typing (with optional static typing via TypeScript), and a secure runtime, eliminating entire classes of errors common in low-level languages. At the same time, modern engines such as **V8, SpiderMonkey, and JavaScriptCore** apply aggressive optimizations, JIT compilation, and advanced runtime techniques, delivering performance that is more than sufficient for the vast majority of real-world applications, even at large scale.

By transpiling to JavaScript, Neob inherits **instant portability, safety by default, and a gigantic ecosystem**, while simultaneously fixing JavaScript‚Äôs historical weaknesses ‚Äî such as semantic inconsistencies, dangerous implicit coercions, and poorly designed APIs. Neob acts as a **disciplined software engineering layer**, enforcing clear rules, compile-time checks, and consistent abstractions that make code predictable, readable, and easy to maintain.

This approach allows beginners to learn programming and software engineering without fighting subtle language pitfalls, while experienced developers gain productivity without sacrificing robustness. The result is a modern, pedagogical, and safe language that delivers real-world applications on top of one of the most battle-tested platforms on the planet.

## ü¶Å&nbsp; Zeca, The Neob Mascot

The Neob mascot is the [chimera](https://en.wikipedia.org/wiki/Chimera_(creature)), a mythological creature that emerged in Greece during the 7th century BC. The chimera is characterized by a hybrid appearance of two or more animals (lion, goat, serpent, eagle) and the ability to breathe fire from its nostrils.

The name ZECA is an acronym for "Zero Effort and Cost Abstractions." Neob is a language based on the concepts of *zero-cost abstraction*, and *zero cognitive effort*, in which the resources and abstractions provided are easy to assimilate and do not impose any performance overhead at runtime. The abstractions provided by Neob are compile-time constructs and do not add any additional cost to the final JavaScript code, making it as fast as if you had written the code in JavaScript without the abstractions.
